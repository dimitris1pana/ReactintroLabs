## Getting Started
### Node Installation
https://nodejs.org/en/download

```bash
npm create vite@latest my-first-react-app -- --template react
cd my-first-react-app
npm install
npm run dev
```

### Material ui installation (optional)

https://mui.com/material-ui/getting-started/installation/

### rect Dev tools installation 
https://react.dev/learn/react-developer-tools

### react router dom installation 
React router dom is a popular library for handling routing in React applications. It allows you to create single-page applications with navigation without full page reloads.

```bash
npm install react-router-dom
```
